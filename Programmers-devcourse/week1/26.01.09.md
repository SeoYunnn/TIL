## 💭 CSS의 이해

> ⚙️**CSS(Cascading Style Sheets) : HTML 이나 XML 과 같은 구조화 된 문서(Documents)를 화면, 종이 등에 어떻게 렌더링할 것인지를 정의하기 위한 언어**
> 
> **👉🏻 HTML 태그를 하나하나 꾸며주는 언어**

<br>

HTML에 CSS를 적용하는 방법은 다음과 같이 3가지가 있음

- **인라인(inline) :** HTML 태그 안에 같이 작성함
- **내부 스타일 시트(internal style sheet) :** HTML 문서 안에 같이 작성함
- **외부 스타일 시트(external style sheet) :** HTML 문서 밖에 작성하고 연결함

<br>

### 1.  인라인 스타일

**HTML 요소의 style 속성을 사용해 스타일을 지정**하는 방법

다른 스타일 방법보다 **우선 순위가 높아**, 덮어쓰기가 쉬움

<br>

**But,** 스타일을 수정하려면 모든 HTML 요소를 수정해야 하므로 **유지 보수가 어려워짐**

👉🏻 인라인 스타일을 가능한 사용하지 않는 것이 좋음

```css
<p style="color: darkblue; background-color: white">인라인 스타일</p>
```

<br>

### 2. 내부 스타일 시트

HTML 문서 **내부에 `<style>` 태그를 사용해 스타일을 지정**하는 방법

인라인 스타일보다는 우선 순위가 낮지만, 외부 스타일보다는 우선 순위가 높음

```html
<!DOCTYPE html>
<html>
<head>
  <title>내부 스타일</title>
  <style>
    p {
      color: darkblue;
      background-color: white;
    }
  </style>
</head>
<body>
<p>내부 스타일</p>
</body>
</html>
```

<img width="800" height="500" alt="Image" src="https://github.com/user-attachments/assets/6ab64572-e18f-4450-88ff-414a20c5a53c" />

⌨️ 태그에다가 직접 하나하나 달아준 코드들을 내부 스타일 시트에 적용 `<style></style>`

⌨️ 기존 코드보다 코드가 더 깔끔해짐

<br>

### 3. 외부 스타일 시트 ⭐️

HTML 문서와는 **별개의 파일에서 스타일을 지정**하는 방법

내부 스타일보다는 우선 순위가 낮지만, 인라인 스타일보다는 우선 순위가 높음

또한, 스타일을 한 번만 작성하면 여러 HTML 문서에서 적용할 수 있으므로 **유지 보수가 용이함 (가장 많이 사용)**

<br>

외부 스타일을 적용하려면, 스타일을 지정한 CSS 파일을 HTML 문서에 로드해야 함

이때, `<link>` 태그를 사용해 CSS 파일 로드할 수 있음

- `<link>` 태그는 보통 `<head>` 태그 안에 위치하며, `href` 속성을 사용해 CSS 파일의 경로를 지정함

- `login.html`

    ```html
    <!DOCTYPE html>
    <html>
        <!-- 외부 스타일 시트 -->
        <head>
            <meta charset="UTF-8">
            <title> LOGIN </title>
            <link rel="stylesheet" href="login.css">
        </head>
        <body>
            <h1>Login</h1>
            <form>
                ID : <input class="login_inputs" type="text">
                <br>
                PW: <input class="login_inputs" type="password">
                <br>
                <br>
                <input id="btn_login" type="button" value="login">
            </form>
        </body>
    </html>
    ```

  ⌨️ `<!— —>` : HTML 주석

<br>

- `login.css`

    ```css
    h1 {
        color: darkblue;
        text-align: center;
    }
    
    .login_inputs { /* 클래스 */
        font-size: 25px;
    }
    
    #btn_login { /* 아이디 */
        font-size: 20px;
        width: 100px;
        height: 30px;
    }
    ```

  ⌨️ `/* */` : CSS 주석

<br>

### 3. 내부 스크립트 & JS

JavaScript에서는 특정 태그를 찾을 때 다음과 같은 기능을 이용함

**1️⃣ id 로 찾기 :** `document.getElemetById(”아이디”)`

**2️⃣ class 이름으로 찾기 :** `document.getElementsByClassName(”클래스 이름”)`

**3️⃣ tag 이름으로 찾기 :** `document.getElementsByTagName(”태그 이름”)`

<br>

```jsx
// ID 란에 입력된 값을 팝업 창에 띄우기
function popId() {
  let userId = document.getElementById("txt_id").value;

  if(!userId) {
    // = userId == ""

  } else {
    alert(userId);
  }
}
```
