## 💭 웹의 이해

> ⚙️ **월드 와이드 웹 (World Wide Web) : 인터넷에 연결된 컴퓨터를 통해 사람들이 정보를 공유할 수 있는 공간**
>
> **👉🏻 간단하게 WWW, W3, 웹이라 불림**

### 1. 웹의 특징

> ⚙️ **웹은 인터넷 상에서 텍스트나 그림, 소리 영상 등과 같은 멀티미디어 정보를 “하이퍼 텍스트(Hyper-Text)” 방식으로 연결하여 제공함**

*   **하이퍼 텍스트(Hyper-Text) :** 문서 내부에 또 다른 문서를 연결되는 참조를 집어넣음으로써 웹 상에 존재하는 웹 페이지끼리 서로 참조할 수 있는 기술
*   웹의 하이퍼 텍스트는 `HTML`로 표현됨
    *   👉🏻 웹의 구조를 담당하는 `HTML`에 더해, 꾸며주는 `CSS`, 동작을 처리하는 `JS`까지 사용하여 구성
*   웹에는 수많은 **“웹 사이트(Website)”** 들이 있고, 각각의 웹사이트는 **“웹 페이지(Web Page)”** 로 구성됨
    *   👉🏻 서로 연관된 내용으로 작성된 **웹 페이지들의 집합**을 **“웹 사이트(Website)”** 라 부름

<br>

### 2. 웹 구조

> ⚙️ ***클라이언트(Client) : 서비스를 이용하는(*요청*) 컴퓨터***
>
> ***서버(Server) : 서비스를 제공하는(*응답*) 컴퓨터***

<img width="786" height="189" alt="Image" src="https://github.com/user-attachments/assets/a4f24527-060f-4fbc-9471-960d461204ab" />

*   **Frontend**
    *   웹 서비스에서 사용자 측면의 그래픽 사용자 인터페이스
    *   사용자와 상호작용(글자 입력, 버튼 클릭, 화면 출력 등)을 담당함
*   **Backend**
    *   웹 서비스에서 사용자의 눈에 보이지 않는 서버 측 영역
    *   프론트엔드에서 전달받은 데이터와 요청을 내부 데이터와 연산을 활용해 처리하고 프론트엔드에 결과를 전달함

<br>

✔️ 인터넷으로 연결된 클라이언트와 서버는 웹 프로토콜인 **HTTP(HyperText Transfer Protocol)** 를 사용해 데이터를 주고받음

<br>

### 3. 간단한 문법 설명

*   `<!DOCTYPE html>` : HTML5 버전으로 작성되었음을 알리는 선언문
*   `<html>…</html>` : HTML 문서의 시작과 끝을 알리는 요소. 모든 태그(head, body)는 이 태그 안에 포함되어야 함
*   `<head>…</head>` : 문서의 메타데이터(인코딩, 제목 등)를 포함하는 부분
*   `<meta charset=”UTF-8”>` : 문자 인코딩 방식 설정
*   `<title>…</title>` : 브라우저 탭 창에 표시되는 제목
*   `<h2>…</h2>` : 제목을 나타내는 태그 (`<h1>`~`<h6>` 중 두 번째 크기)
*   `<a href=”…”>…</a>` : 하이퍼링크를 만드는 태그
*   `<br>` : 줄바꿈 태그
